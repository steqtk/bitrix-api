Дано:
Инфоблок, содержащий одно текстовое свойство (PROPERTY_TEXT), один список (PROPERTY_IS_TURNED_ON) c одним элементом и одна картинка.
Хайлоад блок с одним текстовым свойством (UF_TEXT) и одним чекбоксом (UF_ACTIVE)

Требуется сделать методы:
- POST /api/v1/iblock/add - добавляет элемент инфоблока, заполняя указанные в задании поля
- GET /api/v1/iblock/:id - выводит информацию по элементу инфоблока с заданным id или возвращает ошибку если элемента нет
- Тоже самое повторить для хайлоад блока:
POST /api/v1/highload/add
GET /api/v1/highload/:id

-------------------------------------------------------------------------------------------------------------------------------------

* адреса POST и GET запросов соответствуют ТЗ тестового  задания.
* файл urlrewrite.php прилагается.
* test.postman_collection.json содержит примеры запросов для Postman.
